C File Handling - Creating Files

This page explains how to create and write to files in C programming using fopen() and related functions.

1. Opening a File for Writing

Use fopen() with mode "w" to create a new file or overwrite an existing one.

#include <stdio.h>

int main() {
    FILE *fp = fopen("example.txt", "w");
    if (fp == NULL) {
        printf("Error opening file!\n");
        return 1;
    }
    // Write text to file
    fprintf(fp, "Hello, file!\n");
    fclose(fp);
    return 0;
}

Explanation:

  • fopen("example.txt", "w") opens the file for writing (creates if not exists).
  • fprintf() writes formatted text to the file.
  • fclose() closes the file.

2. Checking if File Opened Successfully

if (fp == NULL) {
    printf("Error opening file!\n");
    return 1;
}

This prevents the program from crashing if the file cannot be opened.

3. Example: Create and Write Multiple Lines

#include <stdio.h>

int main() {
    FILE *fp = fopen("data.txt", "w");
    if (fp == NULL) {
        printf("Could not create file.\n");
        return 1;
    }

    fprintf(fp, "Line 1: Welcome to Hiro Academy!\n");
    fprintf(fp, "Line 2: Learning C file handling.\n");
    fclose(fp);
    printf("File created and written successfully.\n");
    return 0;
}

Output:

File created and written successfully.

4. File Modes Summary

  • "r" — Open for reading (file must exist)
  • "w" — Open for writing (creates new or overwrites)
  • "a" — Open for appending (creates if not exists)
  • "r+" — Open for reading and writing
