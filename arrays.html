C Programming - Arrays

An array in C is a collection of elements of the same data type stored in contiguous memory locations. Arrays are useful for storing multiple values under a single variable name.

1. Declaring an Array

data_type array_name[size];

Example:

int numbers[5]; // array of 5 integers
Note: Array size must be a positive integer constant.

2. Initializing an Array

You can initialize arrays at the time of declaration.

int numbers[5] = {10, 20, 30, 40, 50};

If you donâ€™t specify all values, remaining elements are set to 0.

int nums[5] = {1, 2}; // remaining 3 elements = 0

3. Accessing Array Elements

Array indexing starts from 0.

#include <stdio.h>

int main() {
    int numbers[5] = {10, 20, 30, 40, 50};
    printf("First element: %d\n", numbers[0]);
    printf("Last element: %d\n", numbers[4]);
    return 0;
}

Output:

First element: 10
Last element: 50

4. Looping Through an Array

#include <stdio.h>

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    for (int i = 0; i < 5; i++) {
        printf("arr[%d] = %d\n", i, arr[i]);
    }
    return 0;
}

Output:

arr[0] = 10
arr[1] = 20
arr[2] = 30
arr[3] = 40
arr[4] = 50

5. Memory Representation

Index:   0     1     2     3     4
Value:  10    20    30    40    50
Addr: 1000  1004  1008  1012  1016  (for int, size = 4 bytes)
Note: Memory addresses increase by the size of the data type.

6. Types of Arrays

Type Description Example
One-dimensional Single row of elements int a[5];
Two-dimensional Table-like structure (rows & columns) int a[3][4];
Multi-dimensional Array with more than 2 dimensions int a[2][3][4];

7. Example: 2D Array

#include <stdio.h>

int main() {
    int matrix[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };

    for (int i = 0; i < 2; i++) {
        for (int j = 0; j < 3; j++) {
            printf("%d ", matrix[i][j]);
        }
        printf("\n");
    }
    return 0;
}

Output:

1 2 3
4 5 6
