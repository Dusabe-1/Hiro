C -Output

This page explains how to display output in C programming using the printf() function.

1. Basic Text Output

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Output:

Hello, World!

The \n creates a new line after the text.

2. Printing Variables

#include <stdio.h>

int main() {
    int age = 20;
    printf("I am %d years old.\n", age);
    return 0;
}

Output:

I am 20 years old.

%d is a format specifier for integers.

3. Format Specifiers

Common format specifiers used in printf():

  • %d → Integer
  • %f → Floating-point number
  • %c → Single character
  • %s → String (text)
#include <stdio.h>

int main() {
    int num = 42;
    float price = 9.99;
    char grade = 'A';
    char name[] = "Ellie";

    printf("Number: %d\n", num);
    printf("Price: %.2f\n", price);
    printf("Grade: %c\n", grade);
    printf("Name: %s\n", name);
    return 0;
}

Output:

Number: 42
Price: 9.99
Grade: A
Name: Ellie

%.2f means show the float with 2 decimal places.

4. Printing Multiple Values

#include <stdio.h>

int main() {
    int a = 5, b = 10;
    printf("a = %d, b = %d\n", a, b);
    return 0;
}

Output:

a = 5, b = 10

Multiple variables can be printed in one statement by matching each variable to a format specifier.

5. Special Characters in Output

Escape sequences let you control formatting:

  • \n → New line
  • \t → Tab space
  • \\ → Backslash
  • \" → Double quote
#include <stdio.h>

int main() {
    printf("Line1\nLine2\n");
    printf("Name:\tEllie\n");
    printf("She said, \"Hello!\"\n");
    return 0;
}

Output:

Line1
Line2
Name: Ellie DUSABE
She said, "Hello!"
