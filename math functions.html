C Math Functions

This page explains common math functions in C provided by math.h library.

1. Including <math.h>

To use math functions, include the header:

#include <math.h>

2. Common Math Functions

  • double sqrt(double x); — Square root
  • double pow(double base, double exp); — Power function
  • double fabs(double x); — Absolute value
  • double ceil(double x); — Round up
  • double floor(double x); — Round down
  • double sin(double x); — Sine (x in radians)
  • double cos(double x); — Cosine (x in radians)
  • double tan(double x); — Tangent (x in radians)

3. Example: Using Math Functions

#include <stdio.h>
#include <math.h>

int main() {
    double x = 9.0;
    double y = 2.0;

    printf("sqrt(%.1f) = %.2f\n", x, sqrt(x));
    printf("pow(%.1f, %.1f) = %.2f\n", x, y, pow(x, y));
    printf("fabs(-5.5) = %.2f\n", fabs(-5.5));
    printf("ceil(3.14) = %.2f\n", ceil(3.14));
    printf("floor(3.99) = %.2f\n", floor(3.99));
    printf("sin(3.14159/2) = %.2f\n", sin(3.14159/2));
    return 0;
}

Output:

sqrt(9.0) = 3.00
pow(9.0, 2.0) = 81.00
fabs(-5.5) = 5.50
ceil(3.14) = 4.00
floor(3.99) = 3.00
sin(3.14159/2) = 1.00
