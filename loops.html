C Programming - Loops

In C, loops allow you to execute a block of code repeatedly based on a condition. There are three main types of loops: for, while, and do while.

1. for Loop

The for loop is used when you know in advance how many times you want to execute a statement or a block of statements.

Syntax:

for (initialization; condition; increment/decrement) {
    // code to be executed
}

Example:

#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        printf("i = %d\n", i);
    }
    return 0;
}

Output:

i = 1
i = 2
i = 3
i = 4
i = 5
  • Initialization: Runs only once at the start.
  • Condition: Checked before each iteration. If false, the loop ends.
  • Increment/Decrement: Changes loop variable after each iteration.

2. while Loop

The while loop is used when you don't know beforehand how many times you want to loop. It runs while the condition is true.

Syntax:

while (condition) {
    // code to be executed
}

Example:

#include <stdio.h>

int main() {
    int i = 1;
    while (i <= 5) {
        printf("i = %d\n", i);
        i++;
    }
    return 0;
}

Output:

i = 1
i = 2
i = 3
i = 4
i = 5

Note: If the condition is false initially, the loop body will not run at all.

3. do while Loop

The do while loop is similar to while but the body executes at least once before checking the condition.

Syntax:

do {
    // code to be executed
} while (condition);

Example:

#include <stdio.h>

int main() {
    int i = 1;
    do {
        printf("i = %d\n", i);
        i++;
    } while (i <= 5);
    return 0;
}

Output:

i = 1
i = 2
i = 3
i = 4
i = 5

Note: Even if the condition is false at the beginning, the loop body will execute once.

4. Infinite Loops

// Example of an infinite loop
#include <stdio.h>

int main() {
    while (1) {
        printf("This will run forever!\n");
        break; // use break to exit loop
    }
    return 0;
}

5. Loop Control Statements

  • break: Terminates the loop immediately.
  • continue: Skips the rest of the current iteration and goes to the next one.
#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        if (i == 3) continue; // skip 3
        if (i == 5) break;    // stop loop
        printf("%d\n", i);
    }
    return 0;
}

Output:

1
2
4
