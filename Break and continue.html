C - break and continue

In C, break and continue are control statements that change the flow of loops. They are mostly used inside for, while, and do while loops.

1. break Statement

The break statement immediately terminates the loop or switch statement it is inside.

Syntax:

break;

Flow:

[ Loop Start ] → [ Condition True? ] → Yes → [ Statement ] 
                              ↓
                        [ break; found? ] → Yes → Exit loop

Example:

#include <stdio.h>

int main() {
    for (int i = 1; i <= 10; i++) {
        if (i == 5) {
            break; // exit loop when i == 5
        }
        printf("%d\n", i);
    }
    return 0;
}

Output:

1
2
3
4
Note: The loop stopped as soon as i became 5.

2. continue Statement

The continue statement skips the rest of the code in the current iteration and jumps to the next iteration.

Syntax:

continue;

Flow:

[ Loop Start ] → [ Condition True? ] → Yes → [ Statement ] 
                              ↓
                        [ continue; found? ] → Yes → Go to next loop check

Example:

#include <stdio.h>

int main() {
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            continue; // skip printing when i == 3
        }
        printf("%d\n", i);
    }
    return 0;
}

Output:

1
2
4
5
Note: When i is 3, the printf is skipped.

3. Using break and continue Together

#include <stdio.h>

int main() {
    for (int i = 1; i <= 10; i++) {
        if (i == 3) {
            continue; // skip 3
        }
        if (i == 8) {
            break; // stop at 8
        }
        printf("%d\n", i);
    }
    return 0;
}

Output:

1
2
4
5
6
7

Statement Effect Common Use
