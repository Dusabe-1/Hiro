C- Variables

In C programming, variables are named storage locations that hold data. The value of a variable can change during program execution.

1. Declaring and Initializing Variables

#include <stdio.h>

int main() {
    int age;          // Declaration
    age = 25;         // Assignment
    int year = 2025;  // Declaration + Initialization

    printf("Age: %d\n", age);
    printf("Year: %d\n", year);
    return 0;
}

Output:

Age: 25
Year: 2025

2. Variable Naming Rules

  • Names must start with a letter or underscore (_).
  • Names can contain letters, digits, and underscores.
  • No spaces or special symbols allowed.
  • Variable names are case-sensitive (age and Age are different).
  • Cannot be a C keyword (e.g., int, return).

3. Data Types for Variables

Data Type Size (Typical) Example
int 4 bytes 42
float 4 bytes 3.14
double 8 bytes 3.14159265
char 1 byte 'A'

4. Constants vs Variables

#include <stdio.h>

int main() {
    const float PI = 3.14; // Constant
    int radius = 5;        // Variable
    float area = PI * radius * radius;

    printf("Area = %.2f\n", area);
    return 0;
}

Output:

Area = 78.50

const makes the value unchangeable after initialization.

5. Variable Scope

Local Variable

#include <stdio.h>

void test() {
    int x = 10; // Local to test()
    printf("x = %d\n", x);
}

int main() {
    test();
    // printf("%d", x); // Error: x not accessible here
    return 0;
}

Global Variable

#include <stdio.h>

int g = 100; // Global variable

void test() {
    printf("g = %d\n", g);
}

int main() {
    test();
    printf("g = %d\n", g);
    return 0;
}

Global variables are declared outside functions and can be accessed anywhere in the program.

6. Best Practices

  • Use meaningful names (e.g., totalMarks instead of tm).
  • Initialize variables before using them.
  • Limit the scope of variables to where they are needed.
  • Avoid using too many global variables.
  • Use const for values that should not change.

7. Example Program Using Variables

#include <stdio.h>

int main() {
    int a = 5, b = 10;   // Integer variables
    float avg;           // Floating point variable
    char grade = 'B';    // Character variable

    avg = (a + b) / 2.0; // Calculate average

    printf("a = %d\n", a);
    printf("b = %d\n", b);
    printf("Average = %.2f\n", avg);
    printf("Grade = %c\n", grade);

    return 0;
}

Output:

a = 5
b = 10
Average = 7.50
Grade = B
