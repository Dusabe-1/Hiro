C Function Parameters Explanation

This page explains how function parameters work in C programming with examples.

1. What Are Function Parameters?

A function parameter is a variable declared in the function definition that accepts data when the function is called.

2. Passing Parameters by Value

#include <stdio.h>

void increment(int x) {
    x = x + 1;
}

int main() {
    int a = 5;
    increment(a);
    printf("a = %d\n", a);
    return 0;
}

Output:

a = 5

The function gets a copy of a, so a is unchanged outside.

3. Passing Parameters by Pointer (Reference)

#include <stdio.h>

void increment(int *x) {
    *x = *x + 1;
}

int main() {
    int a = 5;
    increment(&a);
    printf("a = %d\n", a);
    return 0;
}

Output:

a = 6

Passing the address lets the function modify the original variable.

4. Multiple Parameters

#include <stdio.h>

int add(int x, int y) {
    return x + y;
}

int main() {
    int sum = add(3, 4);
    printf("Sum = %d\n", sum);
    return 0;
}

Output:

Sum = 7

Functions can accept multiple parameters separated by commas.

5. Summary

  • Parameters are variables declared in function definitions.
  • Arguments are actual values passed when calling the function.
  • Passing by value copies data; changes inside do not affect the original.
  • Passing by pointer allows the function to modify the original variable.
