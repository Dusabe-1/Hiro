C - if else Statements

In C, if-else statements are used to make decisions in the program based on conditions. The condition is usually a boolean expression that evaluates to either true (non-zero) or false (0).

1. Syntax of if

if (condition) {
    // code to execute if condition is true
}

If the condition is true, the block inside the braces is executed. If it's false, it is skipped.

2. Syntax of if-else

if (condition) {
    // code if condition is true
} else {
    // code if condition is false
}

3. Syntax of if-else if-else Ladder

if (condition1) {
    // code if condition1 is true
} else if (condition2) {
    // code if condition2 is true
} else {
    // code if none of the above conditions are true
}

4. Example - Simple if

#include <stdio.h>

int main() {
    int number = 10;

    if (number > 5) {
        printf("Number is greater than 5\n");
    }

    return 0;
}
Number is greater than 5

5. Example - if-else

#include <stdio.h>

int main() {
    int age = 16;

    if (age >= 18) {
        printf("You are an adult.\n");
    } else {
        printf("You are a minor.\n");
    }

    return 0;
}
You are a minor.

6. Example - if-else if-else Ladder

#include <stdio.h>

int main() {
    int marks = 85;

    if (marks >= 90) {
        printf("Grade: A+\n");
    } else if (marks >= 75) {
        printf("Grade: A\n");
    } else if (marks >= 50) {
        printf("Grade: B\n");
    } else {
        printf("Grade: F\n");
    }

    return 0;
}
Grade: A

7. Notes

  • Conditions inside if must be enclosed in parentheses.
  • Only the block after the first true condition is executed in an if-else if chain.
  • If you omit braces {} for single-line statements, only that single line will be part of the condition.
