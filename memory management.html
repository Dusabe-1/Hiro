C Memory Management

This page explains how to manage memory dynamically in C using malloc(), calloc(), realloc(), and free().

1. malloc() - Allocate Memory

Allocates a block of memory of given size (in bytes). Returns a pointer to the allocated memory.

#include <stdlib.h>

int *ptr = (int *)malloc(5 * sizeof(int)); // allocate memory for 5 integers
if (ptr == NULL) {
    // handle allocation failure
}

2. calloc() - Allocate and Initialize Memory

Allocates memory for an array of elements and initializes all bytes to zero.

#include <stdlib.h>

int *ptr = (int *)calloc(5, sizeof(int)); // allocate and zero initialize
if (ptr == NULL) {
    // handle allocation failure
}

3. realloc() - Resize Allocated Memory

Changes the size of previously allocated memory block.

ptr = (int *)realloc(ptr, 10 * sizeof(int)); // resize to 10 integers
if (ptr == NULL) {
    // handle allocation failure
}

4. free() - Deallocate Memory

Frees previously allocated memory to avoid memory leaks.

free(ptr);
ptr = NULL; // good practice to avoid dangling pointer

5. Example: Dynamic Array Using malloc() and free()

#include <stdio.h>
#include <stdlib.h>

int main() {
    int n = 5;
    int *arr = (int *)malloc(n * sizeof(int));
    if (arr == NULL) {
        printf("Memory allocation failed.\n");
        return 1;
    }

    for (int i = 0; i < n; i++) {
        arr[i] = i * 10;
    }

    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    free(arr);
    return 0;
}

Output:

0 10 20 30 40
