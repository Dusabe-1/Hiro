C Programming - Functions

In C, a function is a block of code that performs a specific task. Functions help break programs into smaller, reusable parts.

1. Function Syntax

return_type function_name(parameters) {
    // function body
    return value; // if return_type is not void
}
- return_type: Type of value the function returns (int, float, void, etc.)
- function_name: Name of the function (should be descriptive)
- parameters: Data passed to the function (optional)

2. Example: Function with No Parameters

#include <stdio.h>

void greet() {
    printf("Hello, welcome to C programming!\n");
}

int main() {
    greet(); // function call
    return 0;
}

Output:

Hello, welcome to C programming!

3. Example: Function with Parameters

#include <stdio.h>

void printSum(int a, int b) {
    printf("Sum = %d\n", a + b);
}

int main() {
    printSum(5, 7);
    return 0;
}

Output:

Sum = 12

4. Example: Function with Return Value

#include <stdio.h>

int multiply(int x, int y) {
    return x * y;
}

int main() {
    int result = multiply(4, 6);
    printf("Result = %d\n", result);
    return 0;
}

Output:

Result = 24

5. Function Declaration and Definition

You can declare a function before main() and define it later.

#include <stdio.h>

// Function declaration
float areaCircle(float radius);

int main() {
    printf("Area = %.2f\n", areaCircle(3.5));
    return 0;
}

// Function definition
float areaCircle(float radius) {
    return 3.14 * radius * radius;
}

Output:

Area = 38.47

6. Types of Functions

  • Library Functions - Predefined (e.g., printf(), scanf())
  • User-defined Functions - Created by the programmer
