C Programming - Comments

Comments in C are notes written in the code to explain what it does. The compiler ignores them, so they do not affect the program’s execution. They are very useful for making code more readable.

1. Why Use Comments?

  • To explain complex logic for future reference.
  • To make code easier for others to understand.
  • To temporarily disable parts of code during testing.
  • To leave reminders for yourself or other programmers.

2. Types of Comments in C

Single-line Comment (//)

#include <stdio.h>

int main() {
    // This prints a greeting message
    printf("Hello, World!\n");
    return 0;
}

Output:

Hello, World!

Everything after // until the end of the line is ignored by the compiler.

Multi-line Comment (/* ... */)

#include <stdio.h>

int main() {
    /* This is a multi-line comment.
       It can span across multiple lines.
       The compiler ignores all of it. */
    printf("C Programming\n");
    return 0;
}

Output:

C Programming

Multi-line comments start with /* and end with */.

3. Using Comments to Disable Code

#include <stdio.h>

int main() {
    printf("First line\n");

    // printf("This line is disabled and won't run\n");

    printf("Last line\n");
    return 0;
}

Output:

First line
Last line

This is useful when testing or debugging code.

4. Nested Comments Problem

Note: C does not support nested comments.

/* This is a comment
   /* This will cause an error */
   printf("Hello"); 
*/

The above will cause a compilation error. You cannot place a comment inside another comment.

5. Best Practices for Comments

  • Keep comments short and relevant.
  • Don’t state the obvious (e.g., // add 1 to x for x++;).
  • Update comments when the code changes.
  • Use comments to explain why, not just what the code does.
  • For large projects, include a comment at the top of the file describing its purpose.

6. Example: Good Commenting Style

#include <stdio.h>

/* Program: Simple Interest Calculator
   Author: Your Name
   Date: 2025-08-09
   Description: This program calculates simple interest
   using the formula SI = (P * R * T) / 100
*/

int main() {
    float principal = 1000.0; // Principal amount
    float rate = 5.0;         // Annual interest rate in percent
    float time = 2.0;         // Time in years

    // Calculate simple interest
    float SI = (principal * rate * time) / 100;

    // Display the result
    printf("Simple Interest = %.2f\n", SI);
    return 0;
}

Output:

Simple Interest = 100.00
