C - Data Types

In C, a data type tells the compiler what kind of value a variable will store. It also determines the memory size, range, and operations that can be performed.

1. Basic Data Types

Type Size (Typical) Range Example
int 4 bytes -2,147,483,648 to 2,147,483,647 int age = 25;
float 4 bytes ~1.2E-38 to 3.4E+38 float price = 9.99;
double 8 bytes ~2.3E-308 to 1.7E+308 double pi = 3.1415926535;
char 1 byte -128 to 127 (signed) or 0 to 255 (unsigned) char grade = 'A';

2. Signed vs Unsigned

C allows numbers to be either signed (can be negative) or unsigned (only positive).

#include <stdio.h>

int main() {
    signed int a = -10;   // Can store negative
    unsigned int b = 10;  // Cannot store negative

    printf("a = %d\n", a);
    printf("b = %u\n", b);
    return 0;
}

Output:

a = -10
b = 10

3. Short, Long, and Long Long

Type Size (Typical) Example
short int 2 bytes short age = 120;
long int 4 or 8 bytes long population = 7500000000;
long long int 8 bytes long long stars = 9223372036854775807;

4. Type Modifiers

  • signed – Allows negative and positive numbers.
  • unsigned – Only positive numbers (doubles the positive range).
  • short – Uses less memory, smaller range.
  • long – Uses more memory, larger range.

5. Derived Data Types

  • Arrays – Collection of same-type elements.
  • Pointers – Store memory addresses.
  • Structures – Group of variables of different types.
  • Unions – Share memory space between different members.
  • Enumerations – User-defined constants.

6. Example Program with Multiple Data Types

#include <stdio.h>

struct Student {
    char name[20];
    int age;
    float marks;
};

int main() {
    int rollNo = 101;           // Integer
    float percentage = 85.6;    // Float
    double pi = 3.1415926535;   // Double
    char grade = 'A';           // Character
    unsigned int u = 300;       // Unsigned integer

    struct Student s1 = {"Alice", 20, 92.5}; // Structure

    printf("Roll No: %d\n", rollNo);
    printf("Percentage: %.2f\n", percentage);
    printf("Pi: %.10lf\n", pi);
    printf("Grade: %c\n", grade);
    printf("Unsigned: %u\n", u);
    printf("Student: %s, Age: %d, Marks: %.1f\n", s1.name, s1.age, s1.marks);

    return 0;
}

Output:

Roll No: 101
Percentage: 85.60
Pi: 3.1415926535
Grade: A
Unsigned: 300
Student: Alice, Age: 20, Marks: 92.5

7. Best Practices

  • Use the smallest data type that fits the value.
  • Prefer double over float for more precision.
  • Use unsigned only when negative values are not needed.
  • Be aware that sizes can differ between systems (use sizeof() to check).
