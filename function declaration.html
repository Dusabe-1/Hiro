C Function Declarations

This page explains how to declare functions in C programming.

1. What is a Function Declaration?

A function declaration tells the compiler about the function's name, return type, and parameters. It is also called a function prototype.

2. Syntax of Function Declaration

return_type function_name(parameter_list);

Example:

int add(int a, int b);
void printMessage(void);

3. Function Definition vs Declaration

Declaration: Provides function signature to the compiler.

int add(int a, int b);

Definition: Provides the actual body of the function.

int add(int a, int b) {
    return a + b;
}

4. Example: Using Function Declaration

#include <stdio.h>

int add(int a, int b);  // function declaration (prototype)

int main() {
    int result = add(5, 7);
    printf("Result: %d\n", result);
    return 0;
}

int add(int a, int b) {  // function definition
    return a + b;
}

Output:

Result: 12

5. Why Use Function Declarations?

  • Allows calling functions before they are defined.
  • Helps the compiler check for correct function usage.
  • Useful when functions are defined in separate files.
